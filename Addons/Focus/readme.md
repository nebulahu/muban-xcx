##使用方法
```php
{:hook('DisplayFocus', array('group' => 'default', 'label' => 'li'))}
```
###参数说明
* group 调用某组的数据，默认“default”
* label 看示例
* 

###示例
```html
<!-- li 就是上面label参数的值，默认为空 -->
<li><a href=""><img src='aaa.jpg'></a></li>
```
